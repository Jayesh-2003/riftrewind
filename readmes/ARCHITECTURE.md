## API Interactions

### Telegram Bot API
```
Request:  POST /getMe
Response: {username: "YourBotName", ...}

Request:  sendMessage(chatId, text)
Response: Message sent

Request:  setMyCommands([{command, description}, ...])
Response: Commands updated
```

### Riot Games API
```
Request:  GET /riot/account/v1/accounts/by-riot-id/{gameName}/{tagLine}
Response: {puuid, gameName, tagLine}

Request:  GET /lol/match/v5/matches/by-puuid/{puuid}/ids
Response: ["match-id-1", "match-id-2", ...]

Request:  GET /lol/match/v5/matches/{matchId}
Response: {info: {participants: [{stats...}, ...]}, ...}
```

### Groq AI API
```
Request:  POST /openai/v1/chat/completions
Body:     {model: "...", messages: [{role, content}], ...}
Response: {choices: [{message: {content: "roast text"}}]}
```

### MongoDB
```
Operation: findOne({telegram_userId: ...})
Response:  User document or null

Operation: findOneAndUpdate({_id}, {$set: {...}}, {upsert: true})
Response:  Updated document

Operation: createIndex({telegram_userId: 1})
Response:  Index created
```
